1. 설치 및 실행
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[최초 실행]
  '설치_및_실행.bat' 를 더블클릭하세요.
  - Python 없음 → 자동 다운로드 + 설치 (Python 3.13, 약 25MB)
  - PySide6 없음 → 자동 pip 설치 (최초 1회, 2~5분 소요)
  - 설치 완료 후 위젯이 자동 실행됩니다.

[이후 실행]
  '설치_및_실행.bat' 를 더블클릭하거나
  바탕화면에 이 파일의 바로가기를 만들어 두면 편합니다.

[위치]
  위젯은 모니터 오른쪽에 자동 배치됩니다.
  (모니터 2개 이상일 경우 오른쪽 모니터 기준)

[데이터 저장 위치]
  %USERPROFILE%\.productivity_widget\tasks.db  (SQLite)
  → 프로그램을 삭제해도 데이터는 이 경로에 유지됩니다.

2. 기본 단축키

  우클릭         컨텍스트 메뉴 (편집/삭제/완료 등)
  Escape         다이얼로그 닫기
  Ctrl + Enter   다이얼로그 저장 (확인)

3. 태스크 관리
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[태스크 종류]
  ① 과제 / To do (todo)       — 구조화된 업무 항목
  ② 긴급 업무 (urgent)        — 빠른 참고용 단순 목록
  ③ 기타 (misc)               — 자유 메모
  ④ 개인 태스크 (personal)    — 개인 일정/업무 직접 입력

[태스크 추가]
  - Ctrl+N 또는 섹션 하단 [+ 태스크 추가] 버튼 클릭
  - 제목, 설명, 목표, 우선순위, 마감일, 색상 설정 가능

[우선순위]
  높음 (빨강) / 보통 (주황) / 낮음 (초록)

[색상 지정]
  태스크마다 개별 색상 선택 가능
  (기본값: 우선순위 색상 자동 적용)

[마감일]
  - 마감일 설정 시 태스크에 날짜 배지 표시
  - 오늘 마감: 주황색 강조
  - 지난 마감: 빨간색 강조

[완료 처리]
  - 태스크 좌측 체크박스 클릭 → 완료 처리 (취소선 + 흐림 표시)
  - 완료 항목은 섹션 하단으로 자동 이동

[태스크 편집]
  - 항목 더블클릭 → 편집 다이얼로그
  - 우클릭 → 편집 / 삭제 / 완료 처리 선택

[태스크 순서 변경]
  - 드래그 앤 드롭으로 순서 변경 가능 (미완료 항목만)

[로그 기록]
  - 우클릭 → 로그 보기 → 해당 태스크의 작업 메모 추가/편집/삭제
  - 로그에 파일 첨부 가능 (파일 경로 저장)

4. 일정 (Schedule) 관리
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[일정 추가]
  캘린더 하단 일정 섹션의 [+ 일정 추가] 버튼 또는
  캘린더 날짜 우클릭 → 일정 추가

[일정 종류]
  - 일반 일정
  - 연차 / 휴가
  - 교육
  - 기타

[필드]
  - 제목, 시작일, 종료일, 시작시간, 장소, 내용

[편집/삭제]
  - 일정 항목 더블클릭 → 편집
  - 우클릭 → 삭제


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
5. 캘린더
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[기본 동작]
  - 월 단위 캘린더 표시
  - 오늘 날짜 자동 하이라이트 (파란 테두리)
  - 태스크 마감일 있는 날짜: 점(dot) 표시
  - 연구실 일정(iCal) 있는 날짜: 색상 점 표시

[날짜 클릭]
  - 클릭한 날짜에 해당하는 태스크 마감 목록 + 연구실 일정 팝업 표시
  - 날짜 하이라이트만 변경 (스크롤 이동 없음)

[이전/다음 달 이동]
  - 캘린더 상단 ◀ ▶ 버튼

[오늘로 이동]
  - 상단 [오늘] 버튼

[마감일 hover 팝업]
  - 마감일이 설정된 날짜에 마우스를 올리면
    해당 날짜 태스크 목록이 팝업으로 표시됨

6. 오늘의 연구실 일정 (카카오워크 iCal 연동)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[개요]
  카카오워크 팀 캘린더를 iCal(.ics) 형식으로 연동하여
  연구실 전체 일정을 위젯에 표시합니다.

[표시 구조]
  ┌ 오늘의 연구실 일정
  │  | [소장님일정] 내용          ← 파란색, 최상단
  │  | [연차] 홍길동              ← 초록색 (연차/예비군/반차 등)
  │  | [교육] AI 교육             ← 주황색 (시간 없는 일정)
  │  | 회의  10:00-11:00  장소    ← 노란색 (시간 지정 일정)
  └ ▶ 내일 (N건)  ← 클릭하여 내일 일정 펼치기/접기

[색상 분류]
  파란색  — "소장님일정" 포함 → 최상단 고정
  초록색  — 연차, 예비군, 오전반차, 오후반차, 출산휴가, 경조휴가 등
  주황색  — 시간 지정 없는 일반 일정 (종일)
  노란색  — 시간 지정 있는 일정 (시작시간-종료시간  장소 형식)

[높이 조절]
  연구실 일정 영역 하단의 드래그 핸들을 위아래로 드래그하여
  표시 영역 높이를 조절할 수 있습니다. (40px ~ 500px)

[Co-work 모드 토글]
  캘린더 상단 🏢 버튼 → 연구실 일정 표시 On/Off

7. 알림
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[마감일 알림]
  - 매 시간마다 오늘 마감 태스크를 Windows 트레이 알림으로 안내
  - 설정 → 알림 탭에서 활성화/비활성화 가능

[아침 브리핑 알림]
  - 매일 지정 시간(기본 09:00)에 오늘/내일 연구실 일정 요약을 트레이 알림으로 전송
  - 설정 → 알림 탭에서 시간 변경 가능

[개인 일정 15분 전 알림]
  - 지정된 인원의 일정 시작 15분 전에 개별 트레이 알림 발송
  - 설정 → 알림 탭에서 대상 인원 이름 등록/편집


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
8. 설정 (Options)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

우측 상단 ⚙ 버튼으로 설정 다이얼로그 열기.
3개 탭으로 구성됩니다.

[탭 1 — 테마/화면]
  테마 선택:
    다크 (기본) / 블랙 / 라이트 / 네이비 / 그레이 / 포레스트
  투명도:
    슬라이더로 위젯 전체 투명도 조절 (0~100%)
  글자 크기:
    슬라이더로 앱 전체 폰트 크기 조절

[탭 2 — 섹션 설정]
  각 섹션(과제/긴급/기타/일정 등)의 표시 여부를 개별 체크박스로 제어

[탭 3 — 알림]
  마감일 알림 On/Off
  아침 브리핑 시간 설정
  15분 전 알림 대상 인원 이름 등록

9. 업무 내역 파일 연동 (Update works)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[개요]
  'Update works' 폴더에 날짜별 .txt 파일을 작성하면
  위젯이 자동으로 태스크를 읽어와 동기화합니다.

[파일명 형식]
  YYYY.MM.DD.txt   예) 2026.03.31.txt

[파일 내용 형식]
  ──────────────────────────────
  [과제 및 To do list]
  1. 업무 제목
  (탭)내용: 세부 내용 설명
  (탭)목표: 달성 목표
  (탭)마감기한: YYYY-MM-DD

  2. 두 번째 업무
  (탭)내용: ...

  [이번주/차주 긴급 업무]
  1. 긴급 업무 내용
  2. 또 다른 긴급 업무

  [기타]
  1. 메모 제목
  자유 형식 내용
  ──────────────────────────────

[섹션 매핑]
  [과제 및 To do list]  → 과제/To do 섹션
  [이번주/차주 긴급 업무]  → 긴급 업무 섹션
  [기타]  → 기타 섹션

[동기화 방식]
  - 파일 내 항목이 DB에 없으면 자동 추가
  - 이미 있는 항목은 중복 추가하지 않음
  - 위젯에서 직접 추가한 항목(manual)은 파일과 무관하게 유지

[업데이트 트리거]
  - 위젯 상단 새로고침(↻) 버튼 클릭
  - 또는 자동 감지 (파일 변경 시)
