1. 설치 및 실행
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[최초 실행]
  '설치_및_실행.bat' 를 더블클릭하세요.
  - Python 없음 → 자동 다운로드 + 설치 (Python 3.13, 약 25MB)
  - PySide6 없음 → 자동 pip 설치 (최초 1회, 2~5분 소요)
  - 설치 완료 후 위젯이 자동 실행됩니다.

[이후 실행]
  '설치_및_실행.bat' 를 더블클릭하거나
  바탕화면에 이 파일의 바로가기를 만들어 두면 편합니다.

[위치]
  위젯은 모니터 오른쪽에 자동 배치됩니다.
  (모니터 2개 이상일 경우 오른쪽 모니터 기준)
  설정 → 테마 & 화면 탭에서 배치 모니터 변경 가능.

[데이터 저장 위치]
  %USERPROFILE%\.productivity_widget\tasks.db  (SQLite)
  → 프로그램을 삭제해도 데이터는 이 경로에 유지됩니다.

2. 기본 단축키
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Ctrl + N       새 태스크 추가
  Ctrl + F       전체 검색 (할 일·긴급·개인업무·일정)
  Ctrl + T       항상 위에 표시 토글 (Always on Top)
  Ctrl + M       위젯 접기 / 펼치기
  Ctrl + ,       설정 열기
  더블클릭       항목 편집 다이얼로그 열기
  우클릭         컨텍스트 메뉴 (편집/삭제/완료 등)
  Escape         다이얼로그 닫기 / 검색창 닫기
  Ctrl + Enter   다이얼로그 저장 (확인)
  Ctrl + ← / →   캘린더 이전/다음 달 이동
  Home           캘린더 오늘 날짜로 이동

3. 태스크 관리
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[태스크 종류]
  ① 과제 / To do (todo)       — 구조화된 업무 항목
  ② 긴급 업무 (urgent)        — 이번 주 단기 처리 업무
  ③ 기타 (misc)               — 자유 메모
  ④ 개인 태스크 (personal)    — 개인 일정/업무 직접 입력

[태스크 추가]
  - Ctrl+N 또는 섹션 하단 [+ 태스크 추가] 버튼 클릭
  - 제목, 설명, 목표, 우선순위, 마감일, 색상 설정 가능
  - 추가 창 제목으로 어느 섹션에 저장되는지 확인 가능
    (🚨 새 긴급업무 추가 / 👤 새 개인업무 추가 등)

[우선순위]
  높음 (빨강) / 보통 (주황) / 낮음 (초록)

[색상 지정]
  태스크마다 개별 색상 선택 가능
  (기본값: ID 기반 파스텔 색상 자동 배정)

[마감일]
  - 마감일 설정 시 태스크에 날짜 배지 표시
  - 오늘 마감: 주황색 강조
  - 지난 마감: 빨간색 강조

[완료 처리]
  - 태스크 좌측 체크박스 클릭 → 완료 처리 (취소선 + 흐림 표시)
  - 완료된 항목은 하단 "✅ 완료업무" 섹션에서 확인/검색 가능
  - 완료 취소: 완료된 항목 우클릭 → "미완료로"
  - 완료된 항목 더블클릭 → 편집 가능 (확인 후)
  - 완료업무 접기/펼치기(▸/▾)로 내용·목표·파일 상세 확인

[태스크 편집]
  - 항목 더블클릭 → 편집 다이얼로그
  - 우클릭 → 편집 / 삭제 / 완료 처리 선택

[태스크 순서 변경]
  - 드래그 앤 드롭으로 순서 변경 가능 (미완료 항목만)

[일괄 처리 (배치 모드)]
  - 섹션 헤더의 ☐ 버튼 → 일괄 선택 모드 진입
  - 여러 항목 체크 후 일괄 완료 / 일괄 삭제 가능

[파일 첨부]
  - 태스크 추가/편집 다이얼로그에서 "＋ 파일 추가" 클릭
  - 커스텀 파일 선택 창: 폴더 트리, 즐겨찾기, Shift/Ctrl 다중 선택
  - 파일 드래그 & 드롭도 지원
  - 확장자별 아이콘과 파일 크기 자동 표시

[로그 기록]
  - 우클릭 → 로그 보기 → 해당 태스크의 작업 메모 추가/편집/삭제
  - 로그에도 파일 첨부 가능

[정렬]
  - 섹션 헤더 정렬 드롭다운: 기본 / 마감일↑↓ / 우선순위 / 생성일↑↓ / 제목

4. 검색
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[전체 검색]
  - 타이틀바 🔍 버튼 또는 Ctrl+F
  - 검색 범위: 과제, 긴급업무, 개인업무, 일정 관리
  - 제목 / 내용 / 목표 / 일정명 / 장소 포함 검색
  - Esc 또는 ✕ 버튼으로 검색창 닫기

[완료업무 검색]
  - "✅ 완료업무" 섹션을 펼치면 전용 검색창 표시
  - 완료된 항목의 제목, 내용, 목표 검색 가능

5. 일정 (Schedule) 관리
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[일정 추가]
  캘린더 하단 일정 섹션의 [+ 일정 추가] 버튼 또는
  캘린더 날짜 우클릭 → 일정 추가

[일정 종류]
  - 📅 단기 일정  — 단발성 일정
  - 🏖 연차 / 휴가
  - 📚 교육
  - ✈️ 출장

[필드]
  - 제목, 시작일, 종료일, 시작시간, 종료시간, 장소, 내용

[편집/삭제]
  - 일정 항목 더블클릭 → 편집
  - 우클릭 → 삭제

6. 캘린더
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[기본 동작]
  - 월 단위 캘린더 표시
  - 오늘 날짜 자동 하이라이트 (파란 테두리)
  - 태스크 마감일 있는 날짜: 점(dot) 표시
  - 연구실 일정(iCal) 있는 날짜: 색상 점 표시

[날짜 클릭]
  - 클릭한 날짜에 해당하는 태스크 마감 목록 + 연구실 일정 팝업 표시

[날짜 우클릭]
  - 해당 날짜로 일정 추가 / 개인업무 추가

[이전/다음 달 이동]
  - 캘린더 상단 ◀ ▶ 버튼
  - Ctrl + ← / Ctrl + → 키보드 단축키

[오늘로 이동]
  - 상단 [오늘] 버튼 또는 Home 키

[마감일 hover 팝업]
  - 마감일이 설정된 날짜에 마우스를 올리면
    해당 날짜 태스크 목록이 팝업으로 표시됨

7. 오늘의 연구실 일정 (카카오워크 iCal 연동)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[개요]
  카카오워크 팀 캘린더를 iCal(.ics) 형식으로 연동하여
  연구실 전체 일정을 위젯에 표시합니다.

[표시 구조]
  ┌ 오늘의 연구실 일정
  │  | [소장님일정] 내용          ← 파란색, 최상단
  │  | [연차] 홍길동              ← 초록색 (연차/예비군/반차 등)
  │  | [교육] AI 교육             ← 주황색 (시간 없는 일정)
  │  | 회의  10:00-11:00  장소    ← 노란색 (시간 지정 일정)
  └ ▶ 내일 (N건)  ← 클릭하여 내일 일정 펼치기/접기

[색상 분류]
  파란색  — "소장님일정" 포함 → 최상단 고정
  초록색  — 연차, 예비군, 오전반차, 오후반차, 출산휴가, 경조휴가 등
  주황색  — 시간 지정 없는 일반 일정 (종일)
  노란색  — 시간 지정 있는 일정 (시작시간-종료시간  장소 형식)

[높이 조절]
  연구실 일정 영역 하단의 드래그 핸들을 위아래로 드래그하여
  표시 영역 높이를 조절할 수 있습니다. (40px ~ 500px)

[Co-work 모드 토글]
  캘린더 상단 🏢 버튼 → 연구실 일정 표시 On/Off

[iCal URL 설정]
  설정(⚙) → 협업 & iCal 탭 → URL 입력 후 🔄 지금 동기화

  - 설정 → 알림 탭에서 활성화/비활성화 가능

9. 설정 (Options)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

우측 상단 ⚙ 버튼 또는 Ctrl+, 로 설정 다이얼로그 열기.
4개 탭으로 구성됩니다.

[탭 1 — 테마 & 화면]
  테마 선택:
    다크 (기본) / 블랙 / 라떼 (라이트) / 네이비 / 그루박스 / 도쿄 나이트
  투명도:
    슬라이더로 위젯 전체 투명도 조절 (20~100%)
  글자 크기:
    스핀박스로 앱 전체 폰트 크기 조절 (8~18pt, 권장 10~11pt)
  글씨체:
    시스템에 설치된 폰트 목록에서 선택
  창 너비:
    픽셀 단위로 조절 (권장 680~780px)
  창 높이:
    0=자동(화면 96%) 또는 400~2160px 수동 지정
  배치 모니터:
    위젯을 표시할 모니터 선택
  버전 선택 / 업데이트:
    GitHub 전체 릴리즈 목록 조회, 변경 내역 미리보기, 원하는 버전 다운로드

[탭 2 — 섹션 설정]
  각 섹션(달력/과제/긴급/일정/기타/개인업무)의 표시 여부를 체크박스로 제어
  바탕화면 바로가기 만들기
  피드백 보내기

[탭 3 — 알림]
  마감일 알림 On/Off
  자동 시작 관련 안내 (Windows 작업 스케줄러로 관리)

[탭 4 — 협업 & iCal]
  카카오워크 iCal URL 입력 및 동기화 주기 설정

10. 데이터 백업 / 복원
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  우측 상단 💾 버튼 → 데이터 백업/복원 다이얼로그

[백업]
  - DB 백업: tasks.db 파일을 백업 폴더에 날짜별 저장
  - JSON 내보내기: 전체 데이터를 JSON 파일로 저장 (이식성 높음)

[복원]
  - JSON 가져오기: 내보낸 JSON 파일로 데이터 복원
  - 7일마다 자동 DB 백업 (앱 시작 시 자동 실행)

[백업 저장 위치]
  %USERPROFILE%\.productivity_widget\backups\
